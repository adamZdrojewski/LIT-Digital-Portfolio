Abre Planner App

Before Innovation Lab

The planner app before I came to the Innovation Lab was just a side project that I got interested in when Abre came to my school and did an App-A-Thon. I learned PHP and some HTML to start making my idea come to life. I was able to get just the basic functions of adding and removing tasks to work, but I had bigger ideas for it as well. Now the app is my Passion Project for the Innovation Lab.


September

The focus this month was to add the ability to separate the tasks by categories and to add the ability to prioritize the tasks. Adding both of these features required the task storage structure to be redesigned as well. I redesigned the structure of the task storage and recreated the add and remove task functions.


October

This month I added the ability to add a due date to each task. You are now able to select the due date through a date picker in the add task window. I also added the ability to check off the completed tasks. Once a task is checked off you can either delete it or uncheck it. A few small layout changes were also made on the home page.


November

This month I added the ability to edit the tasks that you have created. I also made the add task window more simple and it no longer requires you to input a date and priority upfront. This month I also did a_lot of internal and design fixing. This was everything from adjusting how the home page looked to making sure that the text does not go off the screen if task names are long.
